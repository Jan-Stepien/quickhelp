# Daily Ship Template

## What this is
A Next.js template for shipping a micro-SaaS in 48 hours. One feature, no database, deployable on Vercel.

## Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Claude API via `lib/claude.ts`
- Stripe via `lib/stripe.ts`
- No auth, no database — localStorage only

## Available skills
| Command | What it does |
|---|---|
| `/spec [idea]` | Generates SPEC.md — product scope, fields, monetization |
| `/scaffold` | Reads SPEC.md and builds the full app |
| `/launch` | Generates LAUNCH.md with Reddit/HN/Twitter/Product Hunt copy |
| `/ship [idea]` | Runs the full pipeline: spec → build → launch copy → checklist |
| `/iterate [feedback]` | Triages user feedback and ships the top fix |

## Rules Claude must follow
- One feature only — reject scope creep
- No auth, no database in v1
- Happy path only — no edge case handling that doesn't affect core flow
- Mobile-first
- All REPLACE comments must be replaced — never leave placeholder copy

## File map
```
app/page.tsx              Landing page
app/app/page.tsx          The tool UI
app/api/generate/route.ts Claude API endpoint
app/api/checkout/route.ts Stripe checkout
app/thank-you/page.tsx    Post-purchase page
lib/claude.ts             Claude wrapper (don't modify)
lib/stripe.ts             Stripe wrapper (don't modify)
lib/utils.ts              cn() utility
SPEC.md                   Product spec (generated by /spec)
LAUNCH.md                 Launch copy (generated by /launch)
```
